# KriliCar - Gestion de Location de Voitures

Projet Fullstack basé sur **Spring Boot 3** et **Angular**.

## Architecture du projet
- **Backend** : Spring Boot 3.4.2, Java 21, MySQL, MapStruct, Lombok.
- **Frontend** : Angular (à venir).
- **Docker** : Infrastructure MySQL.

## US-0.1 : Initialisation
- Structure de dossiers Monorepo.
- Configuration du socle Backend avec dépendances JPA, Security et Web.
- Mise en place de MapStruct et Lombok.
